## Technologies used
<hr>
Architecture: MVC framework<br>
- Frontend: JSP, HTML, CSS<br>  
- Backend: Java Servlets (Jakarta EE10)<br> 
Database: MySQL with JDBC<br>  
Application server: Apache Tomcat<br>  
Build and dependency management: Apache Maven<br>  


## Database import manual:
<hr>
1. Open MySQL -> Create a new connection with name "school_management"<br>
2. Import schema.sql into connection -> run to process<br>
3. Modify dp.properties -> change username and password match your MySQL profile<br> 
4. Save and run<br>
